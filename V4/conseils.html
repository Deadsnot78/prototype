Conseils par catégorie

Cuisine 🍽️

🚰 Installer un mousseur sur le robinet
🧼 Utiliser le lave-vaisselle à pleine charge
🥣 Réutiliser l’eau de rinçage pour arroser les plantes
💦 Préférer laver les légumes dans une bassine

Salle de bain 🚿

⏱️ Réduire la durée des douches
🚿 Installer une douchette économe
🧽 Couper l’eau pendant le savonnage
🧴 Privilégier les produits sans rinçage

Toilettes 🚽

🔄 Réutiliser l’eau de pluie pour les toilettes
💧 Installer une chasse d’eau double débit
🚫 Ne pas utiliser les toilettes comme poubelle
🪠 Vérifier les fuites de réservoir

Extérieur / Jardin 🌿

🌧️ Récupérer l’eau de pluie
🪴 Arroser tôt le matin ou tard le soir
🍃 Utiliser du paillage pour limiter l’évaporation
🚿 Nettoyer la voiture avec un seau, pas un jet

Maison / Général 🏠

🛠️ Réparer rapidement les fuites
♻️ Réutiliser l’eau de cuisson pour les plantes
📊 Suivre sa consommation régulièrement
🔧 Fermer les robinets correctement
